# Task PY200_T2

## Check if a Number is Positive, Negative, or Zero

---

##  Description

This Python program determines whether a user-entered number is:

* Positive
* Negative
* Zero

It uses conditional statements (`if`, `elif`, `else`) to classify the number.
If the number is negative, the program also displays its positive value using the `abs()` function.

---

##  Objective

* To understand decision-making statements in Python
* To practice user input handling
* To apply logical comparisons
* To use built-in functions like `abs()`

---

##  Program Logic

1. Take input from the user.
2. Convert the input to a floating-point number.
3. Check:

   * If number > 0 → Print positive message
   * If number < 0 → Print negative message and its positive value
   * Otherwise → Print zero message

---

##  Input Example

```
Enter a number: -5
```

---

##  Output Examples

### Example 1: Positive Number

```
Enter a number: 10
Your entered number 10.0 was positive.
```

### Example 2: Negative Number

```
Enter a number: -8
Your entered number -8.0 was negative, however positive value of your number is 8.0.
```

### Example 3: Zero

```
Enter a number: 0
Your entered number 0.0 was zero.
```

---

##  Concepts Used

* `float()` for numeric input
* Conditional statements (`if`, `elif`, `else`)
* Comparison operators (`>`, `<`)
* `abs()` function
* String formatting using `.format()`
